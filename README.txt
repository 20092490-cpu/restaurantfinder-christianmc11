Restaurant Finder

A restaurant management app built for Android.

Features
- Add, edit and delete favourite restaurants
- Search and filter restaurants by name or address
- Sort restaurants A-Z
- Interactive Google Maps view 
- Data persists between app sessions via JSON in shared preferences

Technical Details
- Built with Kotlin
- RecyclerView for restaurant list
- Google Maps SDK for map functionality
- Repository pattern for data management
- Minimum SDK: 26

How to Use
1. Open the app and press the start button on landing
2. View your saved restaurants on the Favourites tab
3. Press Add Favourite to add a new restaurant
4. Press Edit to edit a restaurant
5. Press Delete to delete a restaurant
6. Use the search bar to filter restaurants
7. Press Sort A-Z to sort restaurants alphabetically
